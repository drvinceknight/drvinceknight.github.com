Game Theory: Theory, Software, Research

2026-07-10

  • Authors: Vincent Knight, James Brown.
  • Date: 2026
  • Details: Released online only

I have finished the first release of my game theory textbook. It covers a breadth of topics, combining existing software tools for computation with a selection of classic and contemporary research papers.

The aim is for the book to be useful in three ways: as a text for an undergraduate game theory course (it is the book I use in my own course at Cardiff), as a resource for researchers wanting to use software to apply game theory to their work, and as an entry point into the cornerstone papers behind each topic.

The most exciting thing about this release is that James Brown (nonzerosum.games) has provided illustrations throughout, including the cover. He has written about the collaboration on his site.
